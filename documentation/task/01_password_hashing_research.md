# Task: Password Hashing Research

Informieren Sie sich über Hashverfahren. Halten Sie die Essenz in der Dokumentation fest.
Informieren Sie sich über Salt and Pepper.
Dokumentieren Sie die Essenz zu den obigen Themen.

- **Goal:** Research secure password hashing methods suitable for the Tresor Application backend, specifically focusing on hashing algorithms, the use of Salt, and the use of Pepper.
- **Topics:**
  - Modern Hashing Algorithms (e.g., BCrypt, SCrypt, Argon2)
  - Salt mechanism and its role in preventing rainbow table attacks.
  - Pepper mechanism and its role in adding an extra layer of security.
  - Integration with Java Spring Boot / Spring Security.
